Palindrome Checker

Problem

The program will determine whether a given string is a palindrome.

Language

Python (This solution can be easily adapted to other languages as well)

Input

The input will be a string containing text, letters, numbers, or a combination of these.

Output

The program will print a message indicating whether the input string is a palindrome or not.

Sample Input

Input Output
race car race car is a palindrome
madam madam is a palindrome
hello hello is not a palindrome
12321 12321 is a palindrome
Race Car Race Car is a palindrome

Constraints

  • The input string can be of any length.
  • The program should treat alphanumeric characters (letters and numbers) case-insensitively (e.g., "Race Car" is considered a palindrome).
  • Non-alphanumeric characters (punctuation, spaces, etc.) should be ignored.
